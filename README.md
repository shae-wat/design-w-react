> Implementing a Design with React

## Install

Clone repository and run:

```sh
$ cd app
$ npm install
```

Make sure you are using node 4 and npm 3.
```sh
$ [nvm](https://github.com/creationix/nvm) use 4
$ npm install -g npm@3
```

You will need to ask collaborators for the following files:
• fs
• config-development

## Development

```sh
$ npm run dev
```
